# SilverLink 복약관리 OCR 파이프라인

> 어르신 복약관리를 위한 약봉투·처방전 OCR 기반 약품 식별 시스템  
> OCR 텍스트를 약품 DB와 대조하고, 함량·단위·성분 검증을 거쳐 안전하게 복약 등록까지 연결하는 기능입니다.

---

## 1. 프로젝트 개요

SilverLink의 복약관리 OCR 기능은 어르신이나 보호자가 약봉투 또는 처방전 이미지를 촬영하면, OCR로 텍스트를 추출하고 해당 텍스트를 약품 데이터베이스와 비교하여 복약 등록에 필요한 약품 후보를 찾아주는 기능입니다.

단순히 OCR 결과를 바로 약품으로 확정하지 않고, 다음 단계를 거쳐 안전하게 판정합니다.

```text
약봉투/처방전 이미지
→ OCR 텍스트 추출
→ 텍스트 정규화
→ 약품 후보 검색
→ 함량/단위/성분 검증
→ 신뢰도 계산
→ 최종 판정
→ 사용자 확인
→ 복약 등록
→ 관리자 alias 승인
→ 검색 품질 개선
```
